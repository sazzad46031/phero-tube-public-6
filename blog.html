Q1 : Discuss the scope of var , let and const

The scope of var variable is the global or function scope.It means variable defined outside the function can be accessed globally. It can be updated and redeclared into the scope. The scope of let variable is only block scoped. It can be updated but can not be redeclared into the scope. The scope of const variable is block scope. It can not be updated or redeclared into the scope.

Q2 : Tell us the use cases of null and undefined

We should not set undefined to any value rather than we should use null. There is 8 ways we can get undefined. variable that is not initialized will give undefined. Function with no return will give undefined. Parameter that is not passed will be undefined. If return has nothing on the right side will return undefined.
